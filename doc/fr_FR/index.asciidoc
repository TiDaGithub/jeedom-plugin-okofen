== ÖkoFEN

=== Description
Plugin permetant le controle et la surveillance de chaudières Pelletronic Okofen.
Attention ce plugins n'est n'y edité par Okofen, ni par Jeedom.
Il utilise le même system de contrôle que l'application Web permetant à distance des chaudière basé sur la regulation Pelletronic
